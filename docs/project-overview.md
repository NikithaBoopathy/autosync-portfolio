# AutoSync Portfolio - Project Overview

## Vision
A cloud-native platform where DevOps professionals can update their portfolio skills instantly.

## Tech Stack
- **Frontend:** React 18 + Tailwind CSS
- **Backend:** AWS Lambda (Node.js)
- **Database:** DynamoDB
- **Auth:** Cognito
- **Hosting:** S3 + CloudFront
- **API:** API Gateway
